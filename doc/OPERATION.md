# 每个指令执行的时候需要干什么

0. 使能RAM输出
1. 保持1的同时启用IR输入
2. 启用PC使能，使得PC+1

## LDA~LDD指令
3. 将RAM地址线复用给IR低四位
4. 保持第二步，开启A~D寄存器输入使能
5. 保持第二三步，使能RAM输出

## ADD、SUB指令
3. 根据指令操作数将ALU输入两端复用给不同寄存器；使能ALU +/-运算标志位
4. 保持第二步，开启第一个寄存器使能，将ALU输出端联通数据总线，使能FR输入
5. 空拍

## STOA~STOD指令
3. 将RAM数据线复用给对应寄存器
4. 使能RAM写入
5. 空拍

## JMP指令
3. 将IR低四位联通PC输入端
4. 使能PC写入
5. 空拍

## JZ指令
3. 将IR低四位联通PC输入端
4. 检查FR输出第2位是否为1，如果为1则使能PC写入，否则不使能
5. 空拍

## JNZ指令
3. 将IR低四位联通PC输入端
4. 检查FR输出第2位是否为0，如果为0则使能PC写入，否则不使能
5. 空拍
